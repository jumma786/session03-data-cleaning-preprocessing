# Data Dictionary

| Column Name | Description |
|---|---|
| employee_id | Unique employee identifier |
| department | Employee department |
| region | Employee work location or region |
| engagement_score | Employee engagement score |
| prior_python_score | Previous Python assessment score |
| course_minutes | Total course learning minutes |
| support_tickets_last_30d | Number of support tickets raised in last 30 days |
| manager_support_score | Manager support evaluation score |
| completion_days | Number of days required to complete training |
| department_encoded | Encoded department feature for machine learning |
| prior_python_score_scaled | Scaled Python assessment feature |
